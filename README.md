# Linear Regression Machine Learning Project

## Overview

This project demonstrates the implementation of Linear Regression, a supervised machine learning algorithm, using Python and Scikit-learn. The notebook walks through the complete machine learning pipeline, from data preparation to model evaluation and visualization.

Two regression problems are explored:

* Student Score Prediction – Predicting student exam scores based on the number of hours studied.
* California Housing Price Prediction – Predicting median house values using multiple housing-related features from the California Housing dataset.

⸻

## Features

* Data loading and preprocessing using Pandas
* Feature and target selection
* Train-test data splitting
* Linear Regression model training using Scikit-learn
* Prediction on unseen data
* Data visualization using Matplotlib
